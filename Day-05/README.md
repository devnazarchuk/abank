## 1. Аналіз поточного стану
Поточний проект (`akshsgaur/AI-LMS`) генерує текстові нотатки (Study Material).
Ми розширимо його, додавши відтворення відео.

## 2. Завдання на день

- [ ] **Компонент відеоплеєра:**
  - Створити `_components/VideoPlayer.jsx`.
  - Використати `react-youtube` або `iframe` для відображення.
  - Додати підтримку збереження прогресу (таймкоди), якщо можливо.

- [ ] **Оновлення структури даних:**
  - Додати поле `videoId` (YouTube ID) до таблиці `chapters` або `course_list`.
  - Це дозволить AI або користувачу призначати відео для кожного розділу.

- [ ] **UI Інтеграція:**
  - Вставити плеєр на сторінку курсу (`app/course/[courseId]/page.jsx`).
  - Розмістити його перед текстовими матеріалами.

## 3. Результат
Курс буде складатися не тільки з тексту, а й з відео-матеріалів.
