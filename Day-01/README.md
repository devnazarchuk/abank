# День 1: Аналіз конкурентів та визначення УТП

**Дата:** [[2026-01-22]]
**Мета:** Проаналізувати існуючі навчальні платформи, визначити їхні сильні та слабкі сторони, і сформулювати унікальну торгову пропозицію (УТП) для нашого проекту.

---

## 1. Завдання на день

- [ ] **Визначити основних конкурентів:**
  - [ ] Скласти список з 3-5 головних платформ (напр., Coursera, Udemy, Khan Academy, Prometheus, локальні українські платформи).
- [ ] **Проаналізувати функціонал конкурентів:**
  - [ ] Реєстрація та вхід.
  - [ ] Каталог курсів (пошук, фільтри).
  - [ ] Структура курсу (відео, тексти, тести, практичні завдання).
  - [ ] Інструменти взаємодії (форуми, чати, коментарі).
  - [ ] Монетизація (моделі підписки, разові покупки).
  - [ ] Мобільна версія / додаток.
  - [ ] Унікальні функції (гейміфікація, сертифікати, AI-помічники).
- [ ] **Аналіз візуального стилю та UX:**
  - [ ] Оцінити дизайн, легкість навігації.
  - [ ] Зробити скріншоти вдалих та невдалих рішень.
- [ ] **Сформулювати УТП (Унікальна Торгова Пропозиція):**
  - [ ] На основі аналізу визначити, чим наш проект буде виділятися. (Напр., фокус на інтерактивних вправах, інтеграція з конкретними інструментами, унікальна модель подачі матеріалу, сильна спільнота).

## 2. Результат дня

- Створено документ (або таблицю) з порівняльним аналізом конкурентів.
- Сформульовано 2-3 варіанти УТП для обговорення.

## 3. Нотатки

- Особливу увагу звернути на те, як конкуренти інтегрують відео з YouTube. Чи є це просто вбудоване відео, чи є додаткова обв'язка (нотатки, тайм-коди, інтерактивні елементи)?
- Які AI-функції вже існують? (транскрипції, відповіді на питання, персоналізовані плани). Це допоможе зрозуміти, куди рухається ринок.
