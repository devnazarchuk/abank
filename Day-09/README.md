# День 9: Система обговорень (Коментарі)

**Дата:** 30.01.2026
**Мета:** Додати можливість студентам залишати коментарі та запитання до курсів.

---

## 1. Завдання на день

- [x] **Проектування моделі даних для обговорень:**
  - Створено таблицю `discussions`.
  - Поля: `id`, `course_id`, `user_id`, `user_name`, `user_image`, `comment`, `created_at`.

- [ ] **Розробка API для коментарів:**
  - `POST /api/discussions`: створення нового коментаря.
  - `GET /api/discussions?courseId=123`: отримання всіх коментарів для курсу.

- [ ] **Створення UI компонентів:**
  - Розробити секцію коментарів під матеріалами курсу.
  - Список коментарів з аватарами та часом.
  - Форма для відправки нового коментаря.

## 2. Результат дня

- Студенти можуть ставити запитання та ділитися думками прямо на сторінці курсу.

## 3. Нотатки

- Проект використовує Clerk, тому ми автоматично маємо доступ до імені та фото користувача.
- Коментарі роблять навчання інтерактивним та "живим".
