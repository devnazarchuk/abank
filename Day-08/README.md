# День 8: Вправи та практичні завдання

**Дата:** [[2026-01-29]]
**Мета:** Розширити функціонал лекцій, додавши можливість створювати та проходити вправи або тести.

---

## 1. Завдання на день

- [ ] **Проектування моделі даних для вправ:**
  - [ ] Таблиця `Exercises`: `id`, `lecture_id`, `type` (напр., 'quiz', 'coding_task'), `content` (JSON з даними вправи).
  - [ ] Таблиця `UserAnswers`: `id`, `user_id`, `exercise_id`, `answer`, `is_correct`.
  - [ ] Для 'quiz' `content` може містити масив питань та варіантів відповідей.
- [ ] **Розробка API для вправ:**
  - [ ] `POST /api/lectures/:id/exercises`: створення нової вправи для лекції (адмін).
  - [ ] `GET /api/lectures/:id/exercises`: отримання вправ для поточної лекції.
  - [ ] `POST /api/exercises/:id/submit`: ендпоінт для відправки відповіді користувачем. Логіка перевірки відповіді.
- [ ] **Створення UI для проходження вправ:**
  - [ ] Під вікном з відео лекції додати нову вкладку "Вправи".
  - [ ] Розробити компонент для відображення тесту (Quiz). Питання, варіанти відповідей (radio buttons або checkboxes).
  - [ ] Додати кнопку "Перевірити", яка відправляє дані на бекенд і показує результат (правильні/неправильні відповіді).
- [ ] **Створення UI для додавання вправ (базовий):**
  - [ ] На сторінці редагування курсу (для адміна) додати можливість додавати вправи до лекцій.
  - [ ] Простий конструктор форми: поле для питання, поля для варіантів, позначка правильного.

## 2. Результат дня

- Користувач може відкрити вкладку "Вправи" під лекцією і пройти тест.
- Результати тесту перевіряються і відображаються користувачеві.
- Адміністратор має базову можливість створювати тести для лекцій.

## 3. Нотатки

- Почати варто з найпростішого типу вправ — тестів з одним або кількома варіантами вибору. Більш складні типи (введення коду, відкриті питання) можна додати пізніше.
- Зберігання прогресу проходження вправ (`UserAnswers`) є важливим для майбутньої аналітики та гейміфікації.
- "Практичні відео" можна реалізувати як окремий тип лекції, або просто додати відповідний тег/категорію до існуючих лекцій. На цьому етапі основний фокус на інтерактивних вправах.
