# День 6: Дашборд та перегляд курсів

**Дата:** 28.01.2026
**Мета:** Створити зручний інтерфейс для перегляду списків курсів та перегляду окремого курсу.

---

## 1. Виконані завдання

- [x] **Дашборд користувача (Dashboard):**
  - Реалізовано сторінку `/dashboard` з вітальним банером.
  - Список курсів завантажується автоматично для поточного користувача.

- [x] **Відображення курсів:**
  - Кожен курс представлений у вигляді картки з назвою, описом та темою.
  - Реалізовано перехід до детального перегляду курсу за ID.

- [x] **Сторінка курсу та плеєр:**
  - Реалізовано динамічний роут `/course/[courseId]`.
  - Додано інтегрований **YouTubePlayer**, який автоматично підвантажує відео при виборі розділу (якщо відео ID присутнє в базі).

## 2. Результат дня

- Повністю функціональний дашборд.
- Можливість навігації між курсами та перегляду списку розділів кожного курсу.

## 3. Нотатки

- Проект використовує `@clerk/nextjs` для ідентифікації користувача, тому кожен бачить лише свої курси.
- Всі дані (курси, розділи) зберігаються в локальній базі PostgreSQL.
